# walmart-data-clean-analysis
Data Cleaning &amp; Analysis on Walmart Sales Dataset using Python
First I merge the three of walmart files to ones

## Overview
This project focuses on cleaning, preprocessing, and analyzing Walmart weekly sales data to identify key factors affecting sales performance.

The dataset was prepared and analyzed using Python, with the goal of extracting meaningful insights for business decision-making.

---

## Objectives
- Clean and preprocess raw sales data
- Handle missing and inconsistent values
- Analyze weekly sales trends
- Study the impact of holidays and environmental factors
- Generate visual insights for reporting

---

## Dataset Description
The dataset includes Walmart weekly sales data with features such as:
- Weekly_Sales
- Date
- IsHoliday
- Temperature
- Fuel_Price
- Store Size
- MarkDown values

---

## Data Processing Steps
1. Handling missing values
2. Correcting data types (e.g., Date formatting)
3. Removing invalid or inconsistent records
4. Feature grouping and aggregation
5. Data visualization

---

## Analysis Performed
- Sales comparison between holiday and non-holiday weeks
- Sales trend analysis over time
- Temperature-based grouping (Cold / Medium / Hot)
- Store size impact on weekly sales
- Fuel price relationship with sales
